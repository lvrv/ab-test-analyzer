# A/B Test Analyzer

Простой и понятный анализ A/B-теста для junior-аналитика.

**Что делает:**
- Генерирует синтетический датасет (без внешних файлов).
- Считает конверсию, ARPU и AOV по группам A/B.
- Проверяет гипотезы: χ² (конверсия), t-тест (ARPU).
- Строит графики и сохраняет отчёт `outputs/report.json`.

## Запуск
```bash
pip install -r requirements.txt
python main.py
```
Артефакты в `outputs/`.
